# Diwali-Sales-Analysis
 #### Explore Diwali sales patterns: uncover demographic trends and product preferences from an in-depth analysis, guiding targeted marketing strategies.



#### This project delves into an exploratory data analysis (EDA) of Diwali sales data, uncovering insights and solutions for targeted marketing and product diversification strategies. Key findings include:

## technology used
#### The project leverages Python along with libraries such as NumPy, Pandas, Matplotlib, and Seaborn for exploratory data analysis (EDA) to analyze Diwali sales trends.
